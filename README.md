# SpectrumVibes
SpectrumVibes is a high-performance web-based audio visualizer built with HTML5 Canvas and Web Audio API. It features 8 reactive modes (Bars, DNA, Particles...), customizable cyberpunk palettes, and a real-time recording engine. Users can import MP3s, manage playlists via drag-and-drop, and export high-quality WebM videos up to 4K.
